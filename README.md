# Film Portfolio

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/gabebregman/gabebregman.github.io.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gabebregman.github.io
   ```
3. Install the required dependencies (if any):
   ```bash
   # Example, if using npm
   npm install
   ```

## Deployment Instructions
To deploy your film portfolio, follow these steps:
1. Make sure to commit all your changes:
   ```bash
   git add .
   git commit -m "Your commit message"
   ```
2. Push your changes to GitHub:
   ```bash
   git push origin main
   ```
3. Your portfolio should automatically deploy if you have GitHub Pages enabled in the repository settings!
